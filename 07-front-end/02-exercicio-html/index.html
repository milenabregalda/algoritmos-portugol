Motivos pelos quais eu adoro programação:

  • Computadores sempre foram muito legais para mim
  • É uma forma de poder expressar a minha criatividade: escrever código é como escrever uma história em uma folha em branco
  • Consigo testar a minha habilidade de ser paciente quando tenho que resolver bugs
  • É chique dizer que sou desenvolvedora


Passos para entrar no mundo da programação:

  1. Gostar de tecnologia (você vai precisar, é sério)
  2. Gostar de café (se não toma muito, acredite, vai começar a tomar)
  3. Ter resiliência
  4. Saber pesquisar resoluções por conta própria (ajuda muito!)
  5. Comece por aqui em lógica de programação
